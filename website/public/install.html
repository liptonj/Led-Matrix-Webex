Send a factory reset command to clear all saved settings (WiFi, Webex credentials, etc.).

Common Recovery Scenarios:

  • Device won't boot / stuck on logo: Re-flash the Bootstrap firmware
  • OTA download keeps failing: Flash the Main Firmware directly via USB
  • WiFi won't connect / wrong network: Re-flash Bootstrap or use Factory Reset
  • Want to start completely fresh: Re-flash Bootstrap (erases everything)
